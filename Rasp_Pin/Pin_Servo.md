# 🔄 Servo MG996R - Pin Connection

## การต่อสาย

> ⚠️ **สำคัญ**: ต้องใช้ **ไฟเลี้ยงแยก** สำหรับ Servo!  
> ห้ามใช้ไฟจาก Pi โดยตรง และต้องต่อ Ground พ่วงกัน

| สายสี | ต่อกับ | หมายเหตุ |
|-------|--------|----------|
| 🟠 ส้ม/เหลือง (Signal) | Pin 12 (GPIO 18) | สัญญาณ PWM |
| 🔴 แดง (VCC) | ไฟภายนอก (+) 5V-6V | ห้ามใช้ไฟจาก Pi! |
| 🟤 น้ำตาล (GND) | ไฟภายนอก (-) **+ พ่วง** Pin 6/14 | Common Ground |

---

```
│            │           │ (GPIO 18)    │    │             │
│            │           │              │    │             │
│ VCC 🔴     │───────────│──────────────│────│ (+) 5V-6V   │

└────────────┘   └───────│──────────────│────│ (-) GND     │
                         └──────────────┘    └─────────────┘
                              Common Ground ─────────┘
```

## ค่า Pulse Width สำหรับ MG996R

| ค่า | ตัวเลข |
|-----|--------|
| Min Pulse | 0.0005 (0.5ms) |
| Max Pulse | 0.0025 (2.5ms) |
| มุมหมุน | 0° - 180° หรือ -90° - 90° |
