# 🔋 ADS1115 ADC Module - Pin Connection

## การต่อสาย

> ⚠️ **สำคัญ**: ต่อ VCC เข้ากับ **3.3V เท่านั้น!**  
> ห้ามต่อ 5V เพราะไฟสัญญาณ I2C จะเกินและอาจทำลายขา Pi ได้

| ขา ADS1115 | ต่อกับ Raspberry Pi | หมายเหตุ |
|------------|---------------------|----------|
| VDD (VCC) | Pin 1 (3.3V) | ไฟเลี้ยงโมดูล |
| GND | Pin 9 (GND) | กราวด์ |
| SCL | Pin 5 (GPIO 3) | I2C Clock |
| SDA | Pin 3 (GPIO 2) | I2C Data |
| ADDR | GND | ตั้ง Address เป็น 0x48 |
| A0 | แหล่งจ่าย DC (+) | วัดแรงดัน 0-3V |
