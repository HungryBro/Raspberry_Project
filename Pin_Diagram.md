## ตารางสรุปการเชื่อมต่อ

### 🔋 ADS1115 ADC Module
```
ADS1115          Raspberry Pi
┌────────┐      ┌────────────┐
│ VDD    │─────►│ Pin 1 (3.3V)
│ GND    │─────►│ Pin 9 (GND)
│ SCL    │─────►│ Pin 5 (GPIO3)
│ SDA    │─────►│ Pin 3 (GPIO2)
│ ADDR   │─────►│ GND
│ A0     │◄─────│ DC+ ที่จะวัด (0-3V)
└────────┘      └────────────┘
```

### 🔄 Servo MG996R
```
Servo            Raspberry Pi       External Power
┌────────┐      ┌────────────┐    ┌────────────┐
│Signal🟠│─────►│Pin 12(GPIO18)   │            │
│VCC  🔴 │─────►│─────────────│──►│ (+) 5V-6V  │
│GND  🟤 │──┬──►│Pin 6 (GND) │    │            │
└────────┘  │   └────────────┘    │            │
            └────────────────────►│ (-) GND    │
                Common Ground     └────────────┘
```

### 🚗 Motor Driver (TB6612/L298N)
```
Motor Driver     Raspberry Pi       External Power    Motor
┌───────────┐   ┌────────────┐     ┌────────────┐   ┌───────┐
│ PWMA      │──►│Pin 32(GPIO12)    │            │   │       │
│ AIN1      │──►│Pin 11(GPIO17)    │            │   │       │
│ AIN2      │──►│Pin 13(GPIO27)    │            │   │       │
│ STBY      │──►│Pin 1 (3.3V)│     │            │   │       │  ⚠️ ต้องต่อ!
│ VCC(Logic)│──►│Pin 1 (3.3V)│     │            │   │       │
│ VM        │──►│────────────│────►│(+) 6V-12V  │   │       │
│ GND       │─┬►│Pin 6 (GND) │     │            │   │       │
│ OUT A+    │─│─│────────────│─────│────────────│──►│ (+)   │
│ OUT A-    │─│─│────────────│─────│────────────│──►│ (-)   │
└───────────┘ │ └────────────┘     │            │   └───────┘
              └───────────────────►│ (-) GND    │
                  Common Ground    └────────────┘
```
> 💡 **STBY = Standby**: ต้องต่อ HIGH (3.3V) เพื่อให้ Motor Driver ทำงาน!

## ⚡ Common Ground (สำคัญมาก!)

```
                    ┌─────────────────────────────────────┐
                    │           COMMON GROUND              │
                    │                                      │
    Pi GND ─────────┼──────────────────┐                  │
    (Pin 6,9,14)    │                  │                  │
                    │                  ▼                  │
                    │            ┌─────────┐              │
                    │            │ GND BUS │              │
                    │            └────┬────┘              │
                    │         ┌───────┼───────┬───────┐   │
                    │         ▼       ▼       ▼       ▼   │
                    │      Servo   Motor   ADS1115  DC    │
                    │       GND    Power    GND    Power  │
                    │              GND             GND    │
                    └─────────────────────────────────────┘
```

> ⚠️ **GND ของทุกอุปกรณ์ต้องเชื่อมถึงกันหมด!**

## 📋 สรุป Pin ที่ใช้ทั้งหมด

| Pin (Board) | GPIO | อุปกรณ์ | หน้าที่ |
|:-----------:|:----:|---------|---------|
| 1 | 3.3V | ADS1115, Motor Driver | ไฟเลี้ยง Logic |
| 3 | GPIO 2 | ADS1115 | I2C SDA |
| 5 | GPIO 3 | ADS1115 | I2C SCL |
| 6, 9, 14 | GND | ทุกอุปกรณ์ | Common Ground |
| 11 | GPIO 17 | Motor Driver | AIN1 (ทิศทาง) |
| 12 | GPIO 18 | Servo | Signal (PWM) |
| 13 | GPIO 27 | Motor Driver | AIN2 (ทิศทาง) |
| 32 | GPIO 12 | Motor Driver | PWMA (ความเร็ว) |

## 🔌 สายไฟที่ต้องเตรียม

| ประเภท | จำนวน | ใช้กับ |
|--------|:-----:|--------|
| สาย Jumper F-F | 10+ เส้น | เชื่อม Pi กับ modules |
| สาย Jumper M-F | 5+ เส้น | เชื่อม breadboard |
| สายไฟหนา (มอเตอร์) | 4 เส้น | แหล่งจ่ายไฟ + มอเตอร์ |
| แหล่งจ่าย 5V-6V | 1 ชุด | Servo |
| แหล่งจ่าย 6V-12V | 1 ชุด | Motor |
